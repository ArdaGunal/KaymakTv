# TvTaym

## Projenin Amacı
TvTaym, kullanıcıların favori dizilerini takip edebileceği, son izledikleri bölümleri görebileceği ve Trakt.tv veritabanıyla tam senkronize çalışan modern, siyah-beyaz temalı bir dizi takip uygulamasıdır. Kullanıcı dostu arayüzü ve sade tasarımı ile "İzleme Listesi" (Watchlist) deneyimini pürüzsüz hale getirmeyi hedefler.

## Genel Bakış
Uygulama, Expo ve React Native üzerine kurulmuş olup "Expo Router" yapısı sayesinde modüler bir sayfa geçiş sistemine sahiptir. Trakt API entegrasyonu "Device Code" (Cihaz Kodu) yöntemi kullanılarak oldukça güvenli ve kolay bir şekilde sağlanmıştır.
